# TypeScript-merge-with-unit-test

A TypeScript project for merge three array using merge sort and unit test using Jest

## Project Structure
```
TypeScript-merge-with-unit-test/
├── src/
│   └── index.ts              # Main Merge function
├── test/
│   └── test_merge.test.ts    # Unit tests
```
## Installation

Install dependencies:
```bash
npm install
```

## Running Tests

Run the test suite using:
```bash
npm test
```
